Introduction¶

spm1d is a package for one-dimensional Statistical Parametric Mapping.

It uses Random Field Theory to make statistical inferences regarding registered (normalized) sets of 1D measurements.

Download

Source code and datasets can be downloaded from the spm1d Python or MATLAB GitHub sites.

Support

Submit bug reports and questions by creating a “New Issue” at the spm1d Python issues or MATLAB issues GitHub sites.
