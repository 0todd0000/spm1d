    Quick Installation (Python)¶

    1. Install Anaconda

    2. In a Terminal (Linux, Mac) or an Anaconda Command Prompt (Windows):

      conda install -c toddp spm1d
    • Detailed installation (Python)

    Quick Installation (MATLAB)¶

    1. Download source code from spm1d’s MATLAB github site and unzip.

    2. Add folder containing +spm1d (NOT the +spm1d folder itself) AND the spm8 folder to the MATLAB path.

    • Detailed installation (MATLAB)

    Datasets¶

    All datasets can be accessed in spm1d.data. See the example scripts in ./spm1d/examples for details regarding how to access datasets.
