Keyboard shortcuts¶

Landmark mode¶

Key

Description

A

Add landmark at mouse’s X location (only when template is locked)

L

Lock / unlock template

S

Toggle source visibility (unselected sources)

T

Toggle template visibility

Y

Toggle legend visibility

← →

Select previous / next source

Manual mode¶

Key

Description

O

Toggle original (unwarped) source visibility for the selected source

R

Reset selected source

S

Toggle source visibility (unselected sources)

T

Toggle template visibility

Y

Toggle legend visibility

W

Initialize warp (at the current mouse location)

← →

Select previous / next source

ESC

Cancel warp

ENTER

Apply warp
