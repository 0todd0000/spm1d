This folder contains scripts which aim to demonstrate:

1. Fundamentals of numerically validating spm1d's hypothesis
testing procedures

2. The three main (and equivalent) validation approaches
possible in spm1d (in order of increasingly high-level
interface):
- procedural ("proc")
- object-oriented programming ("oop")
- spm1d's validation user interface: spm1d.util.val.ui ("ui")


Scripts are numbered in the order that they are meant to be
executed; later scripts build upon concepts introduced in
earlier scripts.
